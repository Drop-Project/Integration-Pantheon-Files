# Integration-Pantheon-Files
Pantheon Files is a file manager for elementary OS. This plugin adds the possibility to share the selected files to it's context menu.

## Installation
```
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/usr ..
make
sudo make install
```
